v4.3.0

- feat: Shortcuts at the bottom of the Map View can be now sorted through drag and drop, the sort order is saved per character
- feat: The "Gondor" region name in Map View updated to "Gondor/Mordor" for clarity
- feat: Several travel destination vendors (Aralath, Lumgâna, Alahamâr, Semanísi) added as alternative purchase sources for relevant skills
- feat: Corrected reputation requirements for some teleports to match actual in-game unlock requirements (U46.1 changes)
- feat: Navigation panel in Map View height increased from 65px to 68px for better spacing and icon visibility
- feat: Removed the filter in the enable tab, allowing all trained shortcuts to appear in the sort list regardless of type
- feat: Allegiance skills will now show in the navigation panel of the map view
- feat: several changes to underlying code making it more extensible and robust
- fix:  fixed map location coordinates for West Angmar and East Angmar travel points
